利用規約

2026年4月3日 制定

第1条（適用）

  1. 本規約は、TAKUBOX LLC（以下「当社」といいます）が提供するスマートフォン管理デバイス「TAKUBOX」および関連するサービス（以下総称して「本サービス」といいます）の利用に関する条件を定めるものです。
  2. 利用者（以下「ユーザー」といいます）は、本規約に同意のうえ、本サービスを利用するものとします。
  3. 本サービスにおける学習管理（課題設定、面談サポート等）は、当社が提携する学習塾MAPS（所在地：埼玉県春日部市豊町4-6-1）が実施します。

第2条（定義）

本規約において、次の各号に掲げる用語の意義は、当該各号に定めるところによるものとします。

  1. 「本デバイス」とは、当社が提供するスマートフォン管理デバイス「TAKUBOX」をいいます。
  2. 「専用アプリ」とは、本デバイスと連携して利用する、当社が提供するスマートフォンアプリケーションをいいます。
  3. 「連携教材」とは、本サービスに連携するIoTデジタル教材をいいます。
  4. 「利用契約」とは、本規約に基づき当社とユーザーとの間で締結される契約をいいます。
  5. 「学習管理」とは、学習塾MAPSが実施する課題設定、面談サポートその他の学習支援をいいます。

第3条（利用申込み）

  1. 本サービスの利用を希望する方は、当社所定の方法により利用申込みを行うものとします。
  2. 当社は、利用申込みを受けた場合、当社の基準に従い審査を行い、承認の可否を決定します。
  3. 利用契約は、当社が利用申込みを承認した時点で成立するものとします。
  4. 当社は、以下のいずれかに該当する場合、利用申込みを承認しないことがあります。
    • 申込み内容に虚偽、誤記または記入漏れがあった場合
    • 過去に本規約に違反したことがある場合
    • その他当社が不適切と判断した場合

第4条（利用プラン）

  1. 本サービスには、以下のプランがあります。
    • シンプルプラン：本デバイスによるスマートフォン管理機能および専用アプリの利用
    • お試しプラン：シンプルプランの全機能に加え、連携教材の利用、学習塾MAPSによる課題設定（週1日）および2週間に1回の面談サポート
    • スタンダードプラン：お試しプランの全機能において、課題設定が週2日、面談サポートが週1回となるプラン
    • マックスプラン：スタンダードプランの全機能において、課題設定が毎日となるプラン
  2. 各プランの月額料金は、当社が別途定め、ウェブサイト等に掲示するものとします。
  3. ユーザーは、当社所定の方法により、プランの変更を申し込むことができます。

第5条（本デバイスのレンタル）

  1. 当社は、利用契約の成立後、ユーザーに対し本デバイスを無償で貸与します。
  2. 本デバイスの所有権は当社に帰属し、ユーザーに移転しないものとします。
  3. ユーザーは、本デバイスを善良な管理者の注意をもって管理し、以下の行為をしてはなりません。
    • 本デバイスの分解、改造、修理
    • 本デバイスの転貸、譲渡、担保提供
    • 本デバイスの本来の用途以外での使用
  4. ユーザーの故意または過失により本デバイスが破損・故障した場合、ユーザーは当社に対し修理費用または本デバイスの代金相当額を支払うものとします。
  5. 本デバイスの購入を希望される場合は、別途当社にお問い合わせください。購入の場合、月額料金は異なります。

第6条（利用料金および支払い）

  1. ユーザーは、利用するプランに応じた月額料金を、当社が指定する方法により支払うものとします。月額料金には本デバイスのレンタル料が含まれます。
  2. 月額料金は、利用開始日を起算日とし、毎月発生するものとします。
  3. 当社は、1ヶ月前までにユーザーに通知することにより、料金を改定することができるものとします。
  4. ユーザーが料金の支払いを遅延した場合、年14.6%の割合による遅延損害金を支払うものとします。

第7条（学習管理）

  1. お試しプラン、スタンダードプランおよびマックスプランにおける学習管理は、当社が提携する学習塾MAPSが実施します。
  2. 学習管理の内容（課題の範囲、難易度、面談の日程等）は、お子さまの学年・学力等に応じて学習塾MAPSが決定します。
  3. 学習管理に関するお問い合わせ、面談の日程調整その他の連絡は、学習塾MAPSが窓口となります。
  4. 面談の日程変更は、原則として前日までに学習塾MAPSにご連絡ください。

第8条（連携教材）

  1. お試しプラン、スタンダードプランおよびマックスプランでは、本サービスに連携するIoTデジタル教材をご利用いただけます。
  2. 連携教材の利用にはPCが必要です。
  3. 連携教材の著作権その他の知的財産権は、当該教材の提供元に帰属します。
  4. 連携教材の内容・仕様は、予告なく変更される場合があります。

第9条（対応端末）

  1. 本デバイスで管理できるスマートフォンは、iPhoneまたはおサイフケータイ（FeliCa）対応のAndroid端末に限ります。
  2. 専用アプリ（保護者向け）は、iOSおよびAndroidに対応しています。
  3. 本サービスの利用には、ご自宅にWi-Fi環境が必要です。

第10条（禁止事項）

ユーザーは、本サービスの利用にあたり、以下の行為を行ってはなりません。

  1. 法令または公序良俗に反する行為
  2. 当社または第三者の知的財産権、プライバシー権その他の権利を侵害する行為
  3. 本デバイスの分解、改造、リバースエンジニアリング
  4. 本サービスの運営を妨害する行為
  5. 不正アクセスまたはそれに類する行為
  6. 虚偽の情報を登録する行為
  7. 本サービスを第三者に利用させる行為（ユーザーの家族による利用を除く）
  8. その他当社が不適切と判断する行為

第11条（サービスの中断・変更）

  1. 当社は、以下のいずれかに該当する場合、事前の通知なく本サービスの全部または一部を中断することができます。
    • システムの保守・点検を行う場合
    • 火災、停電、天災地変その他不可抗力により本サービスの提供が困難な場合
    • 通信回線等の障害が発生した場合
    • その他当社がやむを得ないと判断した場合
  2. 当社は、ユーザーへの事前の通知をもって、本サービスの内容を変更することができます。

第12条（解約）

  1. ユーザーは、当社所定の方法により、いつでも利用契約を解約することができます。最低利用期間はありません。
  2. 解約の効力は、当月末日をもって生じるものとします。
  3. ユーザーは、解約後14日以内に本デバイスを当社に返却するものとします。返却にかかる送料はユーザーの負担とします。
  4. 解約後14日以内に本デバイスの返却がなされない場合、当社はユーザーに対し本デバイスの代金相当額（税込91,300円）を請求できるものとします。
  5. 解約時において未払いの利用料金がある場合、ユーザーは速やかにこれを支払うものとします。

第13条（当社による解除）

  1. 当社は、ユーザーが以下のいずれかに該当する場合、事前の催告なく利用契約を解除することができます。
    • 本規約に違反した場合
    • 料金の支払いを2ヶ月以上遅延した場合
    • 登録情報に虚偽があることが判明した場合
    • その他利用契約の継続が困難であると当社が判断した場合
  2. 前項による解除の場合も、第12条第3項および第4項の規定を準用します。

第14条（免責事項）

  1. 当社は、本サービスの利用によるお子さまの成績向上、学習習慣の定着その他の効果を保証するものではありません。
  2. 当社は、本サービスの中断、変更、終了によりユーザーに生じた損害について、当社に故意または重大な過失がある場合を除き、一切の責任を負いません。
  3. 当社の賠償責任は、ユーザーが当社に支払った直近3ヶ月分の利用料金の合計額を上限とします。
  4. 天災地変、感染症の流行その他不可抗力に起因する本サービスの履行遅滞または不履行について、当社は責任を負いません。

第15条（個人情報の取扱い）

  1. 当社は、本サービスの提供にあたり取得するユーザーおよびお子さまの個人情報を、当社のプライバシーポリシーに従い適切に取り扱います。
  2. 当社は、学習管理の実施に必要な範囲で、お子さまの学年、学力その他の情報を学習塾MAPSに提供します。
  3. 前項の情報提供について、ユーザーはあらかじめ同意するものとします。

第16条（知的財産権）

  1. 本サービス、本デバイス、専用アプリおよびこれらに関連する一切の知的財産権は、当社または正当な権利者に帰属します。
  2. ユーザーは、本サービスの利用により、当社の知的財産権の使用許諾を得るものではありません。

第17条（規約の変更）

  1. 当社は、必要と判断した場合、ユーザーへの事前の通知をもって、本規約を変更することができます。
  2. 変更後の規約は、当社ウェブサイトに掲載した時点で効力を生じるものとします。
  3. 変更後に本サービスを利用した場合、ユーザーは変更後の規約に同意したものとみなします。

第18条（準拠法および管轄）

  1. 本規約の解釈および適用は、日本法に準拠します。
  2. 本サービスに関する紛争については、さいたま地方裁判所を第一審の専属的合意管轄裁判所とします。
